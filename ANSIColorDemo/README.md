This provides a small example of using the ANSI escape codes in C to make colored text.

To Build:
```gcc colordemo.c -o colordemo```

Running:
This demo does not take any arguments.

[This Cheatsheet](https://gist.github.com/ConnerWill/d4b6c776b509add763e17f9f113fd25b) came in extremely helpful while making this, and has information on how to do a lot more with this
